# Aegis Spec Visual Identity Assets

This zip contains generated visual assets for the Aegis Spec repository and website.

## Files

- `logo/aegis-spec-logo-minimal.png` — clean logo lockup for README/docs.
- `logo/aegis-spec-logo-expanded.png` — richer logo concept with feature cues.
- `banners/aegis-spec-github-readme-banner.png` — wide banner for GitHub README or social preview.
- `hero/aegis-spec-website-hero.png` — landing page hero visual.
- `illustrations/aegis-spec-workflow-diagram.png` — workflow/supporting illustration for docs or site sections.

## Suggested palette

- Charcoal / black background: `#05080B`, `#0B1115`
- Emerald accent: `#00D084`, `#00E676`
- Amber accent: `#FFB000`, `#FFA203`
- Text light: `#F5F7FA`
- Muted text: `#A9B4BE`
